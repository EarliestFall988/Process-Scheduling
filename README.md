# Git Commands:

## Pull From Git Repository
```

git pull

```

## Push To Git Repository
```

    git add .
    git commit -a -m "<your message>"
    git push

```

## Reset Your Changes to the latest saved repository HEAD (a.k.a help my files are not working...)

```

    git reset --hard

```
